Toys Routes


POST

https://projectelya.herokuapp.com:3000/toys

Add a new toy to the data base, with token for this query


{ "name": "name of the toy",
"info": "Info about the toy",
"category": "The categiry of the toy",
"img_url": "image",
"price": how much the toy is cost }

PUT

https://projectelya.herokuapp.com:3000/toys/:idToy

Update a toy in the data base, with token for this query

{ "name": "name of the toy",
"info": "Info about the toy",
"category": "The categiry of the toy",
"img_url": "image",
"price": how much the toy is cost }

DELETE

https://projectelya.herokuapp.com:3000/toys/:idDel

delete a toy from the data base, with token for this query

GET

https://projectelya.herokuapp.com:3000/toys/search?s=two

Search for toy or some word that include the product description



GET

https://projectelya.herokuapp.com:3000/toys

Get a list of the toys that have 12 results per page

GET

https://projectelya.herokuapp.com:3000/toys/cat/Mind-Games

Search for products by category



GET

https://projectelya.herokuapp.com:3000/toys/prices/?min=10&max=40

It is possible to know the products at the prices we want by searching.
According to the example , we Get a list of all the toys priced between 10 and 40 .





Users Routes

POST

https://projectelya.herokuapp.com:3000/users/login

Add a new user to the data base



{ "name": "the name of the user",
"email": "the email of the user",
"password": "the password of the user" }



POST (login)

https://projectelya.herokuapp.com:3000/users/login

User login, if the email and password are correct the server will respond with a token



{ "email": "the email of the user for login",
"password": "the password of the user for login" }
